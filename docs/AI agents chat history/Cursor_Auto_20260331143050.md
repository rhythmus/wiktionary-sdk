**Cursor**

Short answer: **usually no** — Handlebars newlines mostly become collapsed whitespace, not hard line breaks.

- In normal flow (`white-space: normal`), newline/indentation in `.hbs` renders like a single space.
- Actual line breaks come from:
  - block-level elements (`div`, `p`, `li`, etc.)
  - `display: block`
  - explicit `<br>`
  - width constraints causing wrap (especially with `inline-block` chunks)
- `inline-block` can still wrap to the next line as a whole unit when it doesn’t fit, which often *looks* like “Handlebars line breaks”.

If you want maximum control, use Handlebars whitespace trimming:
- `{{~ ... ~}}` around control tags
- reduce literal whitespace/newlines between inline tags in the template

But the main culprit is almost always **CSS display/wrapping behavior**, not Handlebars itself.

Is it passible, in css to have the entire block of text have and indentation(eg. 1em), aka left-padding, *except* for the first-line?

---

**Cursor**


Yes. The usual pattern is a **hanging first line** using padding plus a negative first-line indent:

```css
.entry-block {
  padding-left: 1em;
  text-indent: -1em;
}
```

- `padding-left: 1em` shifts the whole block’s text inward.
- `text-indent: -1em` applies only to the **first line** and pulls that first line back by `1em`, so it lines up with the unindented edge.

So the **first line** sits where you want (flush with the outer text edge), and **wrapped continuation lines** stay indented by `1em`.

**Caveats**

- This is one block (one flow). If you have several sibling blocks, each block’s first line gets the outdent unless you structure so only the outer wrapper has these rules.
- `text-indent` affects the first line of each block child; for multiple paragraphs inside, each `<p>` would repeat the pattern unless you only wrap the whole thing in one element.

**Alternative** (less common): shape with `::first-line` is limited—padding/margin on `::first-line` is not a reliable substitute for the padding + negative `text-indent` trick for this layout.
